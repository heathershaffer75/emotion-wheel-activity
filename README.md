# 🎨 Emotion Wheel Activity

An interactive, trauma-informed **Primary Mental Health (PMH)** group tool designed to help patients identify, understand, and respond to emotions in a structured and engaging way.

---

## 🧠 Purpose

This activity supports:
- Emotional awareness
- Emotional regulation
- Insight development
- Communication and reflection
- Patient engagement in group settings

It is designed to be **PMH-compliant**, avoiding substance-use-specific language while still addressing core behavioral and emotional patterns.

---

## 🎯 Features

- 🎨 16 emotions (core + expanded categories)
- 🌈 Color-coded emotion tiles for visual engagement
- 🧩 “Shades” of each emotion for deeper identification
- 🔍 Underlying emotions (surface vs core)
- 🎭 Common reactions and patterns
- 🛠️ “Try This” coping strategies
- 💬 Built-in discussion questions for group facilitation

---

## 🧑‍⚕️ How to Use in Group

### Option 1: Emotion Exploration
1. Have a patient select an emotion
2. Review each section together
3. Use discussion questions to guide conversation

### Option 2: Check-In Tool
- Ask: “Which emotion fits you today?”
- Use shades to refine responses

### Option 3: Deep Dive
- Focus on:
  - Underlying emotions
  - Reactions vs. healthier responses
  - Coping strategies
